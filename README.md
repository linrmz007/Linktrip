# Linktrip: A travel buddy app

Description:  
------------- 
LinkTrip is a match-making travel buddy app. Users take a personality test and are match with people they scored a high percentage with.

Dependencies:  
------------- 
* Node
* Express
* MongoDB

Installation:
------------- 
* `git clone this repo and cd into client`
* Run `npm install`
* Run `npm start`
